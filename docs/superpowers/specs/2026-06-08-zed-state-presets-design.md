# Zed 状态形态设计方案

## 状态清单

| 状态 | 触发场景 | 持续时间 |
|------|----------|----------|
| **idle** 空闲 | 无交互时的默认状态 | 持续 |
| **offline** 离线 | 网络断开或服务不可用 | 持续 |
| **listening** 倾听 | 用户在说话 | 随用户说话时长 |
| **thinking** 思考 | 收到消息，等待回复 | 1-10 秒 |
| **speaking** 回复 | Zed 正在输出回复 | 随回复长度 |
| **error** 错误 | API 调用失败等异常 | 短暂（自动恢复） |

---

## 各状态形态设计

### 1. idle — 呼吸冥想

> 平静、内敛、有生命力

- **形态**：缓慢流动的球形粒子云，像在呼吸一样微微膨胀收缩
- **粒子形状**：圆形，柔和发光
- **颜色**：青色 #4ecdc4 为主，缓慢向品红过渡再回来
- **动画**：慢速旋转 + 呼吸脉动，噪声幅度小，整体安详
- **情绪关键词**：🧘 静 · 缓 · 流

---

### 2. offline — 沉睡冬眠

> 服务不可用，处于低功耗等待状态

- **形态**：粒子**极度收敛**成一个暗淡的小球或点，像进入休眠。粒子密度高但亮度低
- **粒子形状**：圆形，无辉光
- **颜色**：深灰蓝色 #2a2a4a，偶尔有微弱的青色闪烁（像在尝试重连）
- **动画**：几乎静止，只有极慢的呼吸（几乎察觉不到）。每隔几秒有一次微弱的"心跳"闪烁
- **情绪关键词**：💤 暗 · 微 · 待
- **恢复**：检测到连接恢复时，粒子缓慢扩散并提亮，平滑过渡到 idle

---

### 3. listening — 声纹聚焦

> 专注接收，像耳朵在捕捉声波

- **形态**：粒子向中心聚拢，形成类似**漏斗/碗状**的收敛形态，中心有一个明亮的"焦点"
- **粒子形状**：圆形，边缘带光晕
- **颜色**：品红 #ff6b6b 偏暖，跟随声波实时跳动
- **动画**：粒子随音频频率**脉动扩散**，像心跳一样有节奏感。噪声幅度跟随声音大小实时变化
- **情绪关键词**：👂 聚 · 暖 · 动

---

### 4. thinking — 神经网络

> 高速运算，像在思考和联想

- **形态**：粒子**分散扩散**成不规则的星云状，多个粒子簇快速游走、碰撞、重组。像大脑中神经元在放电
- **粒子形状**：小三角形或六边形，有棱角感
- **颜色**：青 ↔ 品红**快速交替闪烁**，偶尔有亮白色粒子划过（像思维火花）
- **动画**：旋转加快，噪声幅度增大，粒子速度感明显，有一种"沸腾"的感觉
- **情绪关键词**：🧠 散 · 快 · 闪

---

### 5. speaking — 液态流溢

> 在输出，像液体在流动表达

- **形态**：粒子聚合成一个**流畅的液态水滴/水母形态**，从上向下缓慢流淌，有方向感
- **粒子形状**：圆形偏大，柔和融合
- **颜色**：青色到品红的**渐变流光**，像极光一样在表面流动
- **动画**：跟随 TTS 音频波形有节奏地**上下涌动**，整体形态稳定但有流动感
- **情绪关键词**：🌊 流 · 润 · 涌

---

### 6. error — 信号干扰

> 出错了，像信号受到干扰

- **形态**：粒子**碎裂散落**，像玻璃破碎一样向四周飞散再缓慢回收
- **粒子形状**：三角形，尖锐感
- **颜色**：偏红/橙色，偶尔闪烁
- **动画**：快速抖动 + 随机闪烁，像信号不稳。持续 2-3 秒后自动恢复到 idle
- **情绪关键词**：⚡ 碎 · 闪 · 躁

---

## 状态转换逻辑

```
        ┌──────────┐
        │   idle   │ ←──────────── 所有状态结束回到这里
        └────┬─────┘
             │
     ┌───────┼───────────┐
     ▼       ▼           ▼
 offline  listening   thinking
     │       │           │
     │       │           ▼
     │       │       speaking ──→ idle
     │       │
     └───────┴──────────→ error ──→ idle (自动恢复)
```

## 实现要点

### 状态预设系统
每个状态对应一组 `FluidParams` 预设，通过 `setStatePreset(state)` 方法应用。

### 平滑过渡
状态切换时使用 lerp 插值参数，避免突变：
```typescript
// 每帧插值
currentParams = lerp(currentParams, targetParams, 0.05)
```

### 特殊动画
- **listening**: 音频数据驱动粒子脉动
- **thinking**: 添加随机"火花"粒子
- **speaking**: TTS 音频驱动流体涌动
- **offline**: 周期性微弱心跳闪烁
- **error**: 抖动效果 + 粒子碎裂后回收

### 状态检测
- **offline**: 网络状态监听 + API 心跳检测
- **error**: API 调用异常时触发，3 秒后自动恢复
